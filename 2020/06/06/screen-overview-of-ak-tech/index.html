知识点记录

  • 商用显示器技术原理分类
    • 液晶显示器 LCD(Liquid Crystal Display)
    • 有机发光半导体 OLED(Organic Light-emitting Diode)
  • 显示原理
    • 基于通过三基色组合成像素点的光学原理

  • LCD发光原理

    • 三星LCD屏幕工作原理
    • LCD液晶面板是多层结构
    • 发光原理
      • 当屏幕通电时，背光LED会发出白光, 照亮整个显示面板来实现发光
      • 每个物理像素的Pixel由RGB三个颜色的子像素Sub Pixel组成
      • 每个子像素，都是一个可以被单独控制的透光单元
      • 像素投过背光板发出的白光，通过最外层RGB三种颜色的涂层被白光照亮而显示颜色
      • 空过控制三元素子像素的透光量来显示颜色
        • 红，绿，蓝同时达到最高亮度，就会识别成白色
        • 红，绿不透光，只有蓝色透光，就会显示成蓝色
        • 红色和绿色透光，蓝色不透光，就会显示成黄色
        • 三个子像素全部屏蔽掉背光，就是黑色
      • 两层偏振膜来屏蔽掉背光, 偏振膜还可以过滤掉射进来的杂光
      • 两层偏振膜之间有一层液晶层，通过电压能够控制液晶的螺旋结构，改变通过液晶层的光波的光学特性，使光的偏振角度发生变化。旋转的部分是在液晶层，而不是在偏振膜。
      • 每一个子像素都有独立的控制电压的电路

  • LCD的优缺点

    • 优点
      • 技术和产业非常成熟
      • 能够以低廉价格
    • 缺点
      • 没办法显示出纯黑色
        • 虽然经过两层偏振膜，绝大多数光纤已经被过滤掉了，但还是会有少量的光线穿过偏振膜
        • 全黑色的画面，看起来会发灰。
      • 对比度不够大
        • 对比度: 可以理解为最亮的白色和最暗的黑色的比值。
      • 功耗较大
        • 背光板发出来的光线至少有80%的光线被过滤掉了
        • 即使是显示黑色，整块背光板的功率也是恒定的

  • LED电视

    • LED电视就是LCD的面板，只不过背光板用的是LED发光板

  • OLED

    • 依然是通过三基色组成像素的原理
    • 不同于LCD的几点
      • OLED没有背光板，通过每一个像素点自发光
      • OLED不需要两层偏振膜，OLED的偏振膜不是用来过滤颜色的，而是过滤自然光照射到显示面板之后，再反射回人眼的杂光。
      • OLED显示黑色的时候，子像素是完全关闭的，是真的黑, 纯的黑。

  • OLED优缺点

    • 优点
      • 可以做的很薄, 因为不需要背光板和底层的偏振膜
      • 省电，OLED每个像素自发光，不依靠最下面整层的背光板，电能转换为光能效率高，没有太多的热损耗。OLED的手机在黑夜模式下很省电，因为很多像素点不需要点亮。
      • 对比度远远高于LCD、，黑色是纯色的黑
      • 可以做成柔性屏
    • 缺点
      • 成本高
        • 显示面板生产时候对氧气和空气湿度敏感
        • 有机材料对封装和隔绝又特别高
      • 烧屏现象(Screen Burn-in)
        • 物理层面上避免不了自然损耗，也就是寿命问题
        • 影响因素
          • 持续的通电时间
          • 温度
        • 烧屏是无法修复的
        • LCD也有烧屏现象，但是临时性的，关机冷却几个小时会自动复原

  • 屏幕参数

    • 分辨率
      • 定义: 指的是屏幕上水平方向和垂直方向的像素的数量的乘积
      • 衡量的是显示器显示的精细程度
      • 720P,1080P,4K,8K等分辨率
    • PPI
      • Pixel Per Inch，指的是在一英寸的对角线长度内像素的数量
      • PPI越高越好，但是超过一定的值，就超过了人眼观察的极限了
    • 尼特
      • 衡量一块屏幕的亮度
      • 一尼特指的是在一个平方米的范围内一个烛光(candela)的亮度
      • 屏幕的亮度指的是最高亮度
    • HDR
      • High Dynamic Range, 高动态范围
      • 画面中最亮的元素，用最高亮度来显示，低亮度的元素依然能够维持比较低的亮度
      • 支持的尼特越高，一般来说HDR就越高
      • LCD号称支持的HDR，都是比较差的
      • 支持HDR的面板，一般都是自发光的OLED显示面板和量子点QLED面板(QLED就是背光技术不同的LCD面板)
    • 色域
      • 色域指的是一块屏幕能够显示的色彩范围
      • 主流标准: sRGB和Adobe公司的AdobeRGB，苹果三星力推的DCI-P3

    • 可视角度

      • IPS面板碾压其他各类面板
        • 把电极层和液晶层做在了一个平面，可视角度接近180度，但漏光现象比较严重
        • 泛用性比较强，但是相对有点贵
      • TN面板的可视角度是最差的, 但是优点是在于相对便宜，TN面板适合游戏玩家
      • OLED面板有天然的可视角度的优势，是显示原理导致的。
        • OLED的屏幕色彩更加鲜艳，亮度和可对比度非常高，但是相对来说很贵。

    • FPS

      • Frame per second, 一秒钟计算机的显卡产生出的画面的数量
      • FPS不是一个常量，而是根据画面的生成的难度和显卡的性能随时动态改变的
      • 静态下的计算机画面的FPS非常低
    • 刷新率
      • 屏幕刷新率是个硬件指标
      • 刷新率是60hz，屏幕会以每秒60次的固定频率来显示画面
