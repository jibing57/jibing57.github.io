Vmware Fusion中CentOS 7中安装Vmware Tools

安装步骤

  1. 虚拟机状态栏中选择 虚拟机 > 安装Vmware Tools
    期间可能会报CDROM已经被占用等信息，选择还是挂载即可

  2. 打开Terminal手动挂载CDROM, 如果已经挂载，则不需要挂载

  3. 拷贝cdrom内容到/tmp并安装
