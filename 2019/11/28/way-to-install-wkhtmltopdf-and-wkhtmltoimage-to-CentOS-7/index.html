如何在CentOS 7 中安装wkhtmltopdf和wkhtmltoimage

工具说明

wkhtmltopdf和wkhtmltoimage是一个开源的命令行工具，可以将HTML转换为pdf文档和图片。
Github 仓库: wkhtmltopdf/wkhtmltopdf
官网: https://wkhtmltopdf.org/
